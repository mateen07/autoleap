# AutoLeap AI CRM Case Study — Slide Web App

A lightweight, static web app that presents a 10-slide deck with keyboard navigation, overview grid, and fullscreen mode.

## Run locally
```bash
python -m http.server 8080
```
Open: http://localhost:8080

## Host
Use any static hosting provider (GitHub Pages / Netlify / Vercel). Upload the folder contents.

## Customize
Edit `app.js` (the `slides` array).

## Shortcuts
- ← / → : previous / next
- O : overview
- F : fullscreen
- Home / End : first / last
- ? : help
